# Programming in Python Textbook Content

## Section: Python Intro & Syntax
Python is a high-level, interpreted programming language known for readability. Variables are created by assigning values: `x = 5`.
Standard data types include integers (int), decimal numbers (float), strings (str, e.g., "Hello"), and booleans (bool: True/False).
Comments start with `#`. Use `print()` to output values to the console.
Keywords: python, variables, types, print, comments, strings.

## Section: Control Flow & Loops
Conditional statements use `if`, `elif`, and `else` to control which code block executes. Indentation determines scope in Python.
A `while` loop repeats code as long as a condition is True.
A `for` loop iterates over a sequence (like a list or range). `range(5)` generates numbers from 0 to 4.
Keywords: loops, if-else, while loop, for loop, indentation, range.

## Section: Lists & Tuples
A list is a ordered, mutable collection of items, defined with square brackets: `fruits = ["apple", "banana"]`. You can modify, add, or delete elements.
Slicing extracts parts of a list: `my_list[1:3]`.
Tuples are ordered but immutable (cannot be changed after creation), defined with parentheses: `dimensions = (1920, 1080)`.
Keywords: lists, tuples, mutable, immutable, append, slicing, index.

## Section: Dictionaries & Sets
A dictionary stores data in key-value pairs, defined with curly braces: `user = {"name": "Rahul", "age": 16}`. Keys must be unique.
A set is an unordered collection of unique elements, defined like `numbers = {1, 2, 3}`. Useful for removing duplicates.
Keywords: dictionary, key-value, sets, unique, curly braces.

## Section: Functions & Modules
Functions are reusable blocks of code, defined using the `def` keyword. They can accept parameters and return values.
Local variables exist only inside the function. Global variables exist outside.
Modules are python files containing code you can import: `import math` lets you use `math.sqrt()`.
Keywords: def, function, parameters, return, scope, import, module.
